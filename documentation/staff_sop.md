# Mediqliq Global: Institutional SOP

## 1. OPD Workflow
- **Patient Entry:** Capture photo via USB/Built-in Camera.
- **Consultation:** Use S-Pen/Slash commands for ICD-11 coding.
- **Exit:** Default to Digital Receipt (WhatsApp) to save environment.

## 2. IPD & Emergency (STAT)
- **Admission:** No admission without a Financial Folio (Atomic).
- **Emergency:** Use 'ER-Bypass' toggle for immediate bed assignment.
- **STAT Orders:** Acknowledge within 120 seconds via the Intercom.

## 3. Lab Protocols
- **Auto-Sync:** Do not manually enter results for Sysmex/Mindray machines.
- **Critical Alerts:** High/Low values will pulse Red on the Nursing Dashboard.
